This project is created using Qt Creator:

To compile:

1. Open the project with the CMSC.pro project file
2. Compile using MINGW 7.3.0 32 bit

To use:

Loading files:
- Click the load file button to open a dialog so you can open a existing file or generate a random one
- You are able to both hit load to search for the file or manually enter the file location
- from there a dialog shows up so select how many processes will be created from the template and if you want to randomize runtimes
- hit done to load the processes

Starting the simulation
 - load a file with whatever many processes
 - select a optional cycle time (default is 1 milisecond)
 - then hit start, output from the simulator will be in the text area above the input options
